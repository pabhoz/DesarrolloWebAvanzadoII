XMLHttpRequest

¿Que es?

Es una interfaz empleada para realizar peticiones HTTP y HTTPS a servidores Web. Para los datos transferidos se usa cualquier codificación basada en texto, incluyendo: texto plano,XML, JSON, HTML y codificaciones particulares específicas.

Esta interfaz no es propia de JS, de hecho se implementa en lenguajes como C++, C#, JAVA, etc...

¿Entonces XMLHttpRequest es AJAX?

No! Para eso te estoy diciendo que es arriba... Hay que leer

¿Entonces que es AJAX?

JavaScript asíncrono y XML es una técnica de desarrollo web para la creación de aplicaciones interactivas (RIA)

¿Como funciona?

Se realiza la carga de datos en segundo plano sin necesidad de recargar el sitio (realizar nuevamente una petición HTTP explicita), en su lugar se realiza una petición con XMLHttpRequest al servidor y se procesa la respuesta en el cliente.

¿Y el AJAX si goza?

Claro que si! Ya no tenes que estar refrescando tu navegador para que las cosas sucedan. Ajax está basado en estándares abiertos tales como JavaScript y DOM, por eso es posible implementarlo como multiplataforma.
